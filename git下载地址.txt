

Git在Windows 平台上安装：

在 Windows平台上安装 Git 同样轻松，有个叫做 msysGit 的项目提供了安装包，
可以到 GitHub 的页面上下载 exe 安装文件并运行：

安装包下载地址：https://gitforwindows.org/

点击Repository进入Github进行早期版本下载 ；
点击releases找到自己需要的版本下载；
 
 PS：
 2.10.0 支持xp 系统；
 2.10.2 支持win vista/7 以上版本系统；

 
 
2.10.2版本 
https://github.com/git-for-windows/git/releases?after=v2.11.0-rc3.windows.1
2.10.0版本
https://github.com/git-for-windows/git/releases?after=v2.10.2
